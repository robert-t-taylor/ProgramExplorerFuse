# Academic Program Explorer

**Project Description**

This project is a basic implementation of an online academic program explorer for a university. It allows users to explore available programs and filter them.

**Technology Stack**

* **Frontend:** HTML, CSS, JavaScript (using Fuse.JS framework)
* **Backend (assumed):** Cascade index blocks, JSON, etc.
* **Data Format:** Depends, but JSON mock here.

**Key Features**

* **Program Listing:** Displays all available academic programs.
* **Major Filtering:** Allows users to filter programs by various fields, plus a "fuzzy" keyword/text search'

**Future Enhancements**

* **Minors Support:** Expand the filter to include minors.
* **Server-Side Rendering (SSR):** Improve initial page load performance.
* **State Management:** Use a state management library for complex interactions.
* **Accessibility:** Ensure the application is accessible to users with disabilities.
* **Error Handling:** Implement robust error handling for data fetching and user interactions.
* **User Interface:** Enhance the user interface with CSS styling.

**How to Run**

1. **Start the Application:**
   - Run `python -m http.server`
   - Open `http://localhost:8000/programs.html` in a web browser to view the program explorer.
